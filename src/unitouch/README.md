# UniTouch Baseline

Requires [UniTouch](https://github.com/cfeng16/UniTouch) and [ImageBind](https://github.com/facebookresearch/ImageBind).

```bash
python src/unitouch/inference.py \
    --llama_dir <unitouch_weights_dir> \
    --dataset_root dataset/ \
    --output_csv outputs/unitouch_inference.csv
```
